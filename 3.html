Instructions:

  • Cylinder can be moved and rotated using normal mouse pointer.
  • Use the slider to move the plane filter and the z-value will be populated once you start moving the slider.
  • Using the z-value from the slider, you can view the points present in the rectangle plane in cyclinder in side 2D view.
  • Once you start moving slider, you can see brusing applied and cylinder turns to gray.
  • Added Reset Button to Reset the z-value in cyclinder and color will be back as interpolated Purples.
